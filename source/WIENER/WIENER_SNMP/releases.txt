Version 1.2.0.0
---------------

Added functions:
- Support multiple access functions for channel numbers > 32
- Allow to read the error code for the last SNMP operation

Version 1.1.0.0
---------------

Added functions:
- Set SNMP community passwords for read and write access
- Read out the library version number
- Allow logging to stderr or to a file
- Aliases for some functions to make their naming more consistent with the MIB file:
  * getOutputSwitch replaces getChannelSwitch
  * setOutputSwitch replaces setChannelSwitch
  * getOutputStatus replaces getChannelStatus
  * getMultipleOutputStatus replaces getMultipleChannelStatus
  * getMultipleOutputMeasurementTerminalVoltages replaces getMultipleMeasurementTerminalVoltages
  * getMultipleOutputMeasurementCurrents replaces getMultipleMeasurementCurrents

Version 1.0.0.0
---------------
Initial release
